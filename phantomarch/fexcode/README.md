# FexCode — PhantomArch IDE

## Visão Geral

FexCode é o editor/IDE integrado do PhantomArch, baseado em tecnologia open-source
(VS Code OSS / Code-OSS), customizado com:

- **Tema PhantomArch** (cyberpunk neon)
- **FexAI integrada** (assistente IA offline)
- **Extensões pré-instaladas** para gaming/dev
- **Performance otimizada** (menos telemetria, mais rápido)
- **Suporte completo** a todas linguagens

## Arquitetura

FexCode é uma distribuição customizada do VS Code OSS (Code-OSS) com:
- Branding PhantomArch
- Extensões curadas pré-instaladas
- Integração com FexAI via API local (localhost:7860)
- Settings otimizados para development

## Baseado em

- Code-OSS (VS Code open-source)
- Electron
- Monaco Editor

## Features Exclusivas

1. **FexAI Copilot** — Completação de código via IA local (offline)
2. **Phantom Theme** — Tema visual integrado
3. **Quick Deploy** — Publicar apps rapidamente
4. **Game Dev Mode** — Otimizações para Godot, Unity, Unreal
5. **Terminal Integrado** — Com zsh + starship
6. **Android/Windows** — Templates e build tools integrados
